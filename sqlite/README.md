# 常用的三方框架

## Room

谷歌官方推出的框架，强烈推荐！！！





## LitePal

郭霖大佬推出的SQLite三方框架，简单实用！！！





## GreenDao

greenrobot推出一款SQLite三方框架，也是在ROOM没有推出之前比较热门的一个SQLite三方框架！！！





## ObjectBox

ObjectBox是一款非关系型数据库，ObjectBox作为一个 NoSQL 数据库，直接脱离SQLite，整体性能比SQLite要高出很多！！！
